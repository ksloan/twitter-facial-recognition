twitter-facial-recognition
==========================

Node.js app that checks for the existence of faces in Twitter user's profiles

## Installation
1) Install dependencies

    npm install
 
2) Enter Twitter credentials into checker.js

    consumer_key: '...',
    consumer_secret: '...',
    access_token: '...',
    access_token_secret: '...'
    
3) Start the server

    node index.js
